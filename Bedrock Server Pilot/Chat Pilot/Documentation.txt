To use this addon, you first need to edit or replace permissions.json to match the one provided, next in the world, you need to enable Beta Api under experimental settings. You can do this by opening the world in minecraft and enabling it, or with a world editor such as Universal Minercaft Editor, while you have the world open in minecraft add the included behaviour pack to the world. Also provided are 2 files that start the local http server to recieve and log the messages in message_log.txt, one is in python and one is in javascript, use whatever works best for you.
